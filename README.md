This is repo contains all datasets and python code that performed data analysis for all papers on this research series.

The paper 1 (DOI [10.48550/arXiv.2408.07082](https://doi.org/10.48550/arXiv.2408.07082)) (Zeonodo [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.13890179.svg)](https://doi.org/10.5281/zenodo.13890179)
) and paper 2 (DOI ([10.48550/arXiv.2507.05289](https://doi.org/10.48550/arXiv.2507.05289))).

---
**Paper 2** 
You can find all data in folder [controlled](https://github.com/igorregis/LLMSonarQuarkusAnalysis/tree/master/controlled).

This paper performed a quasi-experiment to evaluate the responses given by 8 LLMs and SonarQube to 3 scenarios of interventions in source code snippets.

---
**Paper 1** 
You can find all data in the folders [Shatteredpixeldungeon](https://github.com/igorregis/LLMSonarQuarkusAnalysis/tree/master/Shatteredpixeldungeon),[Shatteredpixeldungeon4o](https://github.com/igorregis/LLMSonarQuarkusAnalysis/tree/master/Shatteredpixeldungeon4o), [quarkus](https://github.com/igorregis/LLMSonarQuarkusAnalysis/tree/master/quarkus) and [quarkus4o](https://github.com/igorregis/LLMSonarQuarkusAnalysis/tree/master/quarkus4o)

This paper performed a comparative study of code quality analysis using LLM, focusing on readability, comparing the results with a reference tool (SonarQube)
